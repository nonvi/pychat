come usare pychat:

creare una stanza 
1)aprire il file server.py
2)inserire l'indirizzo ip della macchina che si vuole usare 
3)inserire la porta da usare


accedere ad una stanza
1)aprire il file client.py
2)inserire l' indirizzo ip della stanza
3)inserire la porta

how to use pychat:

create a room
1) open the server.py file
2) enter the ip address of the machine you want to use
3) insert the port to use


enter a room
1) open the client.py file
2) enter the ip address of the room
3) insert the door
